# MVP Boundaries

Defines what this MVP includes and excludes to keep delivery focused.

## In Scope

- Managed authentication and role-based access.
- Hybrid donation intake (online and offline/manual).
- Donation reconciliation with explicit exception handling.
- Donor dashboard for history, recurring status, and receipts.
- Public recognition consent visibility with auditable history.

## Out of Scope

- Automated social posting, scheduling, or social data extraction.
- Multi-country tax engine.
- Advanced BI analytics.

## Decision Rule

If a request conflicts with this document, treat it as out-of-scope for the current MVP and defer to a future change proposal.
